This should run on any normal setup that can serve HTML pages/view html pages.

To infect a user, click on them (note, processing js seems to have a bug where
it gives the wrong mosuex/y when browser is too small). The mode (total or
partial is determined via the the select box at top.

usercontainer.js, user,js, and total-infection.pde contain the majority of code.
tests.js contains the tests and is using qunit.
